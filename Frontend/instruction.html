How to Play Simon Game

Instructions:

  1. Press any key to start the game.
  2. Watch the sequence of colors that light up.
  3. Repeat the sequence by clicking on the colored blocks.
  4. If you succeed, the sequence will get longer.
  5. Continue repeating the sequence as it gets more complex.
  6. If you make a mistake, the game will be over.
  7. Try to get the highest score possible!
